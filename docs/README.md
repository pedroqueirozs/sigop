## Resumo Executivo

O **SIGOP** é um sistema web para gerenciar objetos perdidos e encontrados dentro de uma faculdade.
Substitui processos informais (quadros de avisos, grupos de WhatsApp) por um fluxo estruturado e rastreável.

**14 entidades · 9+ relacionamentos · Fluxo completo de devolução com evidências e validação**
